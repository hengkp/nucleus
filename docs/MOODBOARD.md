# Moodboard

The generated visual direction is stored in:

- `windows-app/assets/moodboard.png`
- `web/assets/moodboard.png`

The app visual system uses a quiet Windows utility style:

- clean white surfaces
- teal and blue accents for NAS/network identity
- green status dot for connected state
- amber only for warnings and credential conflicts
- square 8 px radius controls and panels
