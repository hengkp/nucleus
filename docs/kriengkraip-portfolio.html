// about

Where microscopy meets machine learning.

A bioinformatician with 7+ years building image- and omics-analysis pipelines in systems pharmacology. My work spans computational pathology, spatially resolved single-cell analysis, multi-omics integration, and cell-cycle systems biology.

I care about methods that translate complex biological data into reproducible, clinically useful insight — containerized, workflow-managed, and runnable on shared HPC so a result can be repeated, audited, and handed off.

// what i work on

Research focus.

Computational pathology

Reading molecular subtype and biomarker status from H&E whole-slide images with pathology foundation models (UNI2, Virchow2, TITAN) and MCMICRO / SCIMAP workflows.

Spatial & single-cell

AI-assisted spatial transcriptomics for sarcoma progression using CosMx single-cell data and reproducible Python/Jupyter pipelines.

Multi-omics integration

Bulk RNA-seq, pathway analysis (GSVA), and network biology (CollecTRI, DoRothEA) to connect transcriptomic state with phenotype.

Cell-cycle systems biology

Chemically tunable FOXM1 biosensors and live-cell tracking to quantify how cell-cycle regulators drive tumorigenesis.

Machine learning & AI

Applied ML for biomedical images and omics with scikit-learn, TensorFlow, and PyTorch — including LLM-assisted analysis pipelines.

Reproducible infrastructure

Nextflow + Slurm orchestration on AWS and on-prem HPC, with storage and user management for research labs at scale.
